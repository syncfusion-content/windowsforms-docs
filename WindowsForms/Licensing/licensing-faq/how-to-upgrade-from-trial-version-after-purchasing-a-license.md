# How to upgrade from Trial version after purchasing a license

To upgrade from trial version, there are two possible solutions.

1. Uninstall the trial version and install the fully licensed build from the [License & Downloads](https://www.syncfusion.com/account/downloads) section of our website.
2. If you are using Syncfusion controls from [nuget.org](https://www.nuget.org/packages?q=syncfusion), replace the currently used trial license key with a paid license key that can be generated from the [License & Downloads](https://www.syncfusion.com/account/downloads) section of our website. Register the new paid key in your project as described in [How to Register Syncfusion License Key in Windows Forms Application](https://help.syncfusion.com/windowsforms/licensing/how-to-register-in-an-application).

N> License registration is not required if you reference Syncfusion assemblies from a Licensed installer. These licensing changes apply to all evaluators who reference the Syncfusion assemblies from the evaluation installer and those who use Syncfusion NuGet packages from [nuget.org](https://www.nuget.org/).
